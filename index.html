dat2LaTeX

No more boredom with tabular in LaTeX!

Download this project as a .zip file Download this project as a tar.gz file

Welcome to dat2LaTeX.sh webpage.

This is a little bash script to convert a text-only .dat file into a .tex table.
I've created it for personal purposes, then I noticed it could be interesting for other people.

You can download a tagged stable version from one of the link below:

v1.2 | GitHub

Use the script typing:

$ dat2LaTeX.sh

or

$ dat2LaTeX.sh -h

to read a little help.

Enjoy LaTeX!

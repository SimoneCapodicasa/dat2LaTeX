Welcome to dat2LaTeX.sh webpage.

This is a little bash script to convert a text-only .dat file into a .tex table.

You can type:

$ dat2LaTeX.sh

or

$ dat2LaTeX.sh -h

to read a little help to use this script properly.

Enjoy LaTeX!
